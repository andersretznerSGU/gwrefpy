GWREFPY
=======



Installation
------------
The package is uploaded to the Python Package Index (PyPI) and can be installed using pip.

Install the package:

.. code-block:: console

    pip install gwrefpy

Update the package to the latest version:

.. code-block:: console

    pip install gwrefpy --upgrade

Uninstall the package:

.. code-block:: console

    pip uninstall gwrefpy
